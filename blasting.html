    Sistem ini mendeteksi bahwa Supabase Anda menolak pembuatan grup baru karena tabel contacts masih dikunci oleh Check Constraint bawaan (biasanya bernama contacts_category_check).

    Langkah Perbaikan (Hanya 1 Menit):

    1. Login ke Dashboard Supabase.
    2. Buka menu SQL Editor (Ikon di menu kiri).
    3. Klik New Query, lalu salin dan jalankan (Run) kode SQL persis di bawah ini:
    ALTER TABLE contacts DROP CONSTRAINT IF EXISTS contacts_category_check;

    Setelah menjalankan SQL di atas, silahkan kembali ke sini dan ulangi pembuatan/drag grup Anda!
